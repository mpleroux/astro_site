# README

A project to build a personal website using Astro. This will eventually include a blog, a photo gallery, and web development projects among other things.

URL: [mleroux.me](https://mleroux.me)

## Technologies Used

- [Astro](https://astro.build/) for the lightweight CMS
- [Tailwind CSS](https://tailwindcss.com/) for the CSS
- [Netlify](https://www.netlify.com/) for deployment
