# README

A project to build a personal website using Astro. This will eventually include a blog, a photo gallery, and web development projects among other things.

URL: [mleroux.me](https://mleroux.me)

## Services Used

| Service | Description |
| -------- | ----------- |
| [Astro](https://astro.build/) | Lightweight web framework |
| [Tailwind CSS](https://tailwindcss.com/) | CSS utility classes |
| [Netlify](https://www.netlify.com/) | Deploys the application |
